Information about %PATH% variable

On Windows, %PATH% environment variable determines which directories are searched to find an application whose name was typed from the command line

The value is a string that consists of name of directories separated by ";" character.

There are 2 %PATH% variables:

  • system one, which applies to all users
  • per-user, which is combined with the system one, but is different for each user

System variable is stored in the registry under HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\Environment.

User variable is stored in the registry under HKCU\Environment.

HKLM stands for HKEY_LOCAL_MACHINE.

HKCU stands for HKEY_CURRENT_USER.
